How to add a new tool card

📖 Adding a new card — 3 easy steps

01 Open home.html in any text editor (Notepad, VS Code, etc.).
Find the comment that says TEMPLATE: Copy this block to add a new tool
02 Copy the template <a class="card"> block.
Paste it above the template comment. Change the href to your new page filename, update the emoji, title, description, tag, and pick a colour for --card-color.
Colour ideas: #00ffc8 teal  ·  #ff6b35 orange  ·  #7c6fff purple  ·  #ffd166 yellow  ·  #06d6a0 green
03 Save the file and upload it to your GitHub repo (featte.github.io/ims_playdoh/).
GitHub Pages updates automatically within ~1 minute. Done! ✅
